
# REtech svelte kit app starter / boilerplate

- typescript
- flowbite
- tailwind
- flowbite icons

## Creating a project

Use the template button on github to start a new project

- create .env
- setup src/lib/colors.js
- run `yarn`
- run `yarn dev`
